Um programa para treinar lógica de programação

Banco de dados para sistema de aprendizado baseado em raciocínio.

Tabelas principais:
- problems: armazena os problemas
- hints: dicas progressivas (núcleo do sistema)
- user_attempts: histórico do usuário (futuro)

Diferencial:
O sistema não fornece respostas diretas, mas guia o usuário através de etapas de raciocínio.
